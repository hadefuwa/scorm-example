How It Works

Two halves: the engineer prepares the source, then the AI processes it.

The workflow is easiest to understand when human preparation and AI automation are shown as two separate, sequential parts of the same pipeline.

What The Engineer Does

Prepare the source so the AI has something safe and structured to work from.

1

Write in Word

Create the course, homework, worksheet, and assessment content in one source document.

One master authoring file
2

Add tags

Mark blocks with tags like <HTML>, <worksheet>, and <document>.

The AI knows what output type each block should become
3

Add filenames

Use <filename> tags so each block has a clear target name.

Outputs land in the right folders with the right names
4

Define SCORM decisions

Add version, launch, tracking, completion, and navigation details if SCORM packaging is required.

Packaging happens safely instead of being guessed
5

Save into the repo

Place the prepared file in the source folder for the pipeline to use.

AI has a defined, well-structured input to process

What The AI Does

Five sequential stages from source to checked output.

Analyse

Read the source

In Tagged DOCX

Finds all tagged blocks, validates filenames, identifies warnings, and notes any missing SCORM decisions.

Out reports/analysis-report.md
Extract

Pull out valid blocks

In Analysed source blocks

Creates clean extracted text files from each valid tagged block, stripping wrapper tags and teacher notes.

Out outputs/extracted/
Build

Create learner outputs

In Extracted block files

Builds learner-facing HTML pages and worksheet or document outputs from the clean extracted content.

Out outputs/html/ & outputs/docx/
Validate

Check the result

In Built outputs

Checks for missing files, broken references, duplicates, and any unresolved items before declaring success.

Out reports/validation-report.md
Package

Attempt SCORM if safe

In Validated outputs + SCORM decisions

Packages only if launch file, version, tracking, and navigation details are all explicitly defined. Otherwise stops safely.

Out SCORM package or safe stop
