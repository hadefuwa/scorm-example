Design brief

Design a program that controls two sets of traffic lights at a road junction.

Hardware

You will need:

  • Upstream board
  • Combo board — connected on Port A and Port B

The traffic light outputs are mapped as follows:

Port Bit Signal Set of lights
Port ABit 0GREENJunction A
Port ABit 1AMBERJunction A
Port ABit 2REDJunction A
Port BBit 0GREENJunction B
Port BBit 1AMBERJunction B
Port BBit 2REDJunction B

Software

Create a panel in Flowcode with the two sets of lights in the appropriate colour. You can use the single LED component for each light.

You can carry out this program in simulation and then try it on the hardware in the next lesson.

Design the program so that the lights cycle through this sequence:

RED 20 seconds
AMBER 3 seconds
GREEN 20 seconds

✓  When there is RED at one set of lights there must be GREEN at the other.
